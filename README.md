SSR，全称为 ShadowsocksR（下文简称为SSR） 来历：

ShadowsocksR是breakwa11发起的Shadowsocks分支，在Shadowsocks的基础上增加了一些数据混淆方式，称修复了部分安全问题并可以提高QoS优先级。后来贡献者Librehat也为Shadowsocks补上了一些此类特性，甚至增加了类似Tor的可插拔传输层功能。

ShadowsocksR开始时曾有过违反GPL、发放二进制时不发放源码的争议，不过后来ShadowsocksR项目由breakwa11转为了与Shadowsocks相同的GPL、Apache许可证、MIT许可证等多重自由软件许可协议。

2017年7月19日，ShadowsocksR作者breakwa11在Telegram频道ShadowsocksR news里转发了深圳市启用SS协议检测并被大量用户转发，引发恐慌。7月24日，breakwa11发布了闭源的SS被动检测程序，引发争议。 2017年7月27日，breakwa11遭到自称 “ESU.TV” 的不明身份人士人身攻击，对方宣称如果不停止开发并阻止用户讨论此事件将发布更多包含个人隐私的资料，随后breakwa11表示遭到对方人肉搜索并公开个人资料的是无关人士，为了防止对方继续伤害无关人士，breakwa11将删除GitHub上的所有代码、解散相关交流群组，停止ShadowsocksR项目。但项目已被多人fork。并有人在其基础上继续发布新的版本，例如SSRR

Shadowrocket 2.1.13：全面解析与使用指南
Shadowrocket是一款强大的网络代理工具，广泛应用于iOS设备，帮助用户访问被限制的网站与服务。最新版本2.1.13在功能上有了一些显著的增强，本文将围绕其主要特点、脚本使用、免费与付费选项、VPS配置及美版的差异进行详细探讨。

1. Shadowrocket的基本功能
Shadowrocket主要功能是代理服务器的管理与使用，用户可以通过配置不同的代理节点，快速切换不同的网络环境。其界面简洁，操作直观，适合各类用户，尤其是需要翻墙访问国外内容的用户。

2. Shadowrocket脚本的使用
Shadowrocket支持使用自定义的配置脚本，这些脚本可以极大地简化节点的管理。用户可以在网上找到各种公共脚本，或根据自己的需要进行编写。例如，用户可以添加多种节点并设置不同的规则，以确保在不同情况下自动选择最优的代理服务器。

脚本示例：

plaintext
复制代码
[Proxy]
# 代理节点
Proxy1 = ss, 192.168.1.1, 8388, aes-256-gcm, password, obfs=http
Proxy2 = vmess, 192.168.1.2, 443, username=your_username, password=your_password

[Rule]
# 规则设置
DOMAIN-SUFFIX,example.com,Proxy1
DOMAIN-KEYWORD,google,Proxy2
FINAL,DIRECT
使用这些脚本时，确保每个节点的设置正确，并在Shadowrocket中导入脚本即可实现自动化管理。

3. Shadowrocket的免费与付费版本
Shadowrocket并非完全免费，用户在使用时可能需要为特定的功能或节点付费。尽管有些免费节点可以使用，但通常速度较慢且稳定性差。因此，建议用户考虑购买VPS（虚拟私人服务器），自己搭建稳定的节点，以获得更好的网络体验。

4. VPS搭建与配置
对于希望提升上网速度和安全性的用户，搭建自己的VPS是一种理想的解决方案。以下是搭建VPS的基本步骤：

选择VPS服务商：如DigitalOcean、Vultr或Linode等。
安装代理软件：常见的有Shadowsocks、V2Ray等。
配置服务器：确保设置好端口、加密方式及密码等信息。
在Shadowrocket中添加节点：将VPS的连接信息添加到Shadowrocket中，即可使用。
5. 美版Shadowrocket的特色
美版Shadowrocket相较于其他版本，通常提供更为稳定和高速的代理服务。其设计更为注重用户体验，界面友好，功能完备。用户可以轻松访问Netflix、YouTube等被限制的服务，而不必担心被封锁。

6. Shadowrocket的全局模式
Shadowrocket还支持全局模式，这意味着所有流量都通过设定的代理服务器传输。这在需要完全保护隐私或访问特定内容时非常有用。用户只需在设置中开启全局模式，即可确保所有网络请求都通过代理。
